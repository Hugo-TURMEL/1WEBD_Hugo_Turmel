# CinéPalace - Application Web de Cinéma

Application web vintage permettant de découvrir des films tendances, rechercher des films et consulter leurs détails.

##  Fonctionnalités

-  Page d'accueil avec films en tendance
-  Recherche de films en temps réel
-  Page détails avec synopsis, casting, genres
-  Design responsive mobile-first
-  Design vintage rétro

##  Technologies utilisées

- HTML
- CSS
- JavaScript
- API TMDB (The Movie Database)

## Installation

1. Cloner le dépôt
```bash
git clone https://github.com/Hugo-TURMEL/1WEBD_Hugo_Turmel.git
```

2. Ouvrir `index.html` dans un navigateur
